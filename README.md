## Installation

clone the repository

```bash
$ git clone https://github.com/XaviaTeam/ts-discord-bot-chatgpt.git
```

install dependencies

```bash
$ npm install
```

<hr/>

## Setup

Open .env.example, add channel id from your server, openaikey, token, then save it as .env

```bash
$ mv .env.example .env
```

## Usage

```bash
$ npm run build
$ npm start
```
